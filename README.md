## 📌 What this app does

claude-code-t8 is a desktop app for working with Claude-style chat and code tools on Windows. It gives you a simple way to use local or remote models with a clean desktop flow.

It is built for end users who want to:

- use an OpenAI-compatible API
- connect to Ollama
- run in desktop mode
- work with code and text in one place
- avoid a setup that feels hard to manage

## 🖥️ Before you start

Use a Windows PC with:

- Windows 10 or Windows 11
- at least 4 GB of RAM
- 200 MB of free disk space
- a stable internet connection for download and setup
- admin access if Windows asks for it

For better results:

- use 8 GB of RAM or more
- close extra apps before you launch it
- keep your GPU drivers updated if you plan to use local models

## 🧯 Simple fixes

### The app does not open

- run the file again
- restart Windows
- extract the full zip before opening the app
- move the app to a folder with a short path, like `C:\Apps\claude-code-t8`

### Windows blocks the file

- right-click the file
- open Properties
- check Unblock if you see it
- try again

### The model does not respond

- check your internet connection
- confirm the API key is valid
- make sure the model name is correct
- restart the app
- test with a different model if you have one

### Ollama does not connect

- confirm Ollama is running
- check the host and port
- make sure the model is installed in Ollama
- try `http://localhost:11434` if your setup uses the default port
